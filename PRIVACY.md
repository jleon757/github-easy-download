# Privacy Policy for GitHub Easy Download

Last updated: December 2024

## Overview

GitHub Easy Download is a Chrome extension that adds download buttons to GitHub repository pages. We are committed to protecting your privacy and handling your data responsibly.

## Data Collection

### What we DON'T collect:
- No personal information
- No browsing history
- No user credentials
- No analytics or tracking data
- No data is sent to external servers (except GitHub's public API)

### What we DO store locally:
- **Cache data**: Temporary storage of GitHub API responses to reduce API calls (stored in browser's local storage)
- **User preferences**: Your settings choices (stored in Chrome's sync storage)
  - Include pre-releases option
  - Old release warning preference
  - Cache duration preference

## Permissions Used

### Storage Permission
- Used to cache GitHub API responses locally
- Used to save your preferences
- Data never leaves your device

### Tabs Permission
- Used only to detect which GitHub repository you're viewing
- No browsing history is collected or stored

### Host Permission (api.github.com)
- Used to fetch release information from GitHub's public API
- Only accesses publicly available repository data
- No authentication required

## Third-Party Services

### GitHub API
- We make requests to GitHub's public API to fetch release information
- These are the same requests you would make by visiting the releases page manually
- No authentication tokens are used
- Subject to GitHub's privacy policy

## Data Storage

- All data is stored locally on your device
- Cache data expires automatically based on your settings (default: 15 minutes)
- You can clear all cached data at any time through the extension popup

## Data Sharing

- We do not share any data with third parties
- We do not have access to any of your data
- All processing happens locally in your browser

## Updates

This privacy policy may be updated occasionally. We will notify users of any material changes by updating the "Last updated" date.

## Contact

If you have questions about this privacy policy or the extension, please:
- Open an issue on our GitHub repository
- Contact us through the Chrome Web Store listing

## Compliance

This extension complies with:
- Chrome Web Store Developer Program Policies
- GDPR (General Data Protection Regulation)
- Chrome Extension Manifest V3 security requirements

## Your Rights

You have the right to:
- Clear all cached data at any time (via the extension popup)
- Uninstall the extension at any time
- View all source code (open source)
